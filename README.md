# LeetCode Solutions

This repository is a curated collection of LeetCode problems I have solved to build and demonstrate my problem-solving and coding experience.

The solutions are organized broadly by topic so they are easy to browse by technique rather than just by problem number. My goal is to keep the repository readable, consistent, and useful as both a revision resource and a record of the kinds of algorithmic problems I have worked through.

## Organization

- `graph-traversal/` for BFS, DFS, and graph search problems
- Additional folders will be grouped by major topics such as trees, dynamic programming, intervals, and backtracking

## What Each Solution Includes

- A clear implementation of the core idea
- A short note on the approach
- Time and space complexity where useful

## Purpose

This is not meant to be a full archive of every submission. It is a focused collection of representative solutions that shows breadth across common interview topics and a consistent coding style.
